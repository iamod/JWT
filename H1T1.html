Visual Studio Code asennus yms.

Asennus onnistui lataamalla uusin versio Visual Studio Code -sivuilta.
VSC:ssä on liitännäinen nimeltänsä "Live Server". Tämä liitännäinen toimii paikallisena palvelimena.
Emmetit tulevat automaattisesti kun aloittaa kirjoituksen '<'-merkillä ja halutun osan kirjaimella.
Monikursori toimii pitämällä ALT-näppäintä pohjassa ja napauttamalla vasenta hiirenpainiketta useammassa eri kohdassa.
